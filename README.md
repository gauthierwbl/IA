# Application de Jugement de Popularité Musicale

## Description

Cette application utilise des techniques d'intelligence artificielle prédictive et générative pour évaluer la popularité des morceaux de musique. En se basant sur un dataset de pistes Spotify, l'application propose également des paramètres pour créer des musiques susceptibles de devenir populaires.

## Fonctionnalités

- Évaluation de la popularité des morceaux de musique.
- Suggestions de paramètres pour générer des musiques populaires.
- Interface utilisateur intuitive pour interagir avec l'application.

## Technologies Utilisées

- Python
- Pandas
- Scikit-learn
- Flask / Django (selon le choix final)
- Bibliothèques d'IA générative (comme TensorFlow ou PyTorch)
